Nimbus demo

Nimbus is a Go MapReduce-style system with a coordinator, RPC workers, grep-style map execution, JSONL intermediate output, and lease-based reassignment.

System

The coordinator tracks task state. Workers request one task at a time, load the referenced file locally, run the current grep map logic, write `mr-<task-id>.jsonl`, and report completion back over RPC.
